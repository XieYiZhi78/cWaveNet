# Conditional WaveNet
這篇的內容主要是展示了一個受WaveNet啟發的CNN模型，那它是有針對時間序列預測進行簡化和優化的結構，這篇的作者把它應用在財務相關領域，然後在conditional WaveNet的情況下，能夠提取時間序列之間的時間關係來改進預測，同時也不需要較長時間的歷史資訊，也可以減少noise
